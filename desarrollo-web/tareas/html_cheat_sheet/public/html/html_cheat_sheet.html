<h1> - divisions

division tags are used to divide the content of a website into sections.

the tags described in this page are all divided with divs, for example.

you can't really see them, but they're there! and they provide organization and strucutre for the webpage.

they're also useful when it comes to applying styles with css and functionality with javascript.

<span style="{styles}"> </span> - spans

span tags are used to apply styles to a specific part of a text.

for example, this text is green.

<strong> - strong emphasis

the strong tag is used to, semantically, give strong emphasis to text.

this is particularly useful for when the website is read by a text-to-speech app, for example.

when a text in between a strong tag, it will be spoken in a lower tone when read by these apps, although it visually looks very similar to using regular bold text.

this is an example of text that is strongly emphasized.

<i> - italicized text

The i tag is used to italicize text

you can use for dramatic effect, like this!

<u> - underlined text

the u tag is used to underline text.

this text is underlined, as you can see.

<b> - bold text

the b tag is used to make text bold.

unlike the strong tag, the b tag only makes the text bold visually and does not provide any semantic value to the website.

just like this!

<strike> - strikethrough text

the strike tag is used to strikethrough text. doesn't have a semantic equivalent.

this text has been strikethrough'd.

<small> - small text

The small tag is used to make text smaller.

like this text that is teeny tiny.

<em> - emphasized text

the em tag is used to emphasize text. visually, it looks like simply an italicized text, but it works similarly to the strong tag.

it will give a special emphasis to text when read by text-to-speech apps, and, visually, it looks like an i tag was used.

this is an emphasized text, for example.

<form> - form

the form tag is used to create an HTML form for user input



which has the following strucutre:

<form>

    <label for="name"> Name: </label>

    <input type="text" id="name" name="name">

<br>

    <label for="email" Email:>

    <input type="email" id="email" name="email">

<br>

    <input type="submit" value="Submit">

</form>
